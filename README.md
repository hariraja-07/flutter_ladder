--- 

## Installation 

1. Clone the repository:
    ```bash
    git clone https://github.com/hariraja-07/flutter_ladder.git
    ```

2. Navigate to the project directory:
    ```bash
    cd flutter_ladder
    ```

3. Install dependencies:
    ```bash
    flutter pub get
    ```

4. Run the app:
    ```bash
    flutter run
    ```
